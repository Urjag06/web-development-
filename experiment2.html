HTML Text Formatting Tags Reference

HTML provides several tags for formatting text to improve its appearance and convey semantic meaning. This page lists the most common and currently supported tags.

Semantic and Physical Tags

Tag Description Example Code Rendered Output
`<b>` Defines bold text, without any extra importance (physical tag, for stylistic purposes). `<p>This is <b>bold</b> text.</p>`

This is bold text.
`<strong>` Defines important text (logical tag, adds semantic importance). Typically displayed in bold. `<p>This is <strong>important</strong> text.</p>`

This is important text.
`<i>` Defines a part of text in an alternate voice or mood (physical tag, e.g., technical terms, foreign phrases). `<p>This is <i>italic</i> text.</p>`

This is italic text.
`<em>` Defines emphasized text (logical tag, adds semantic emphasis). Typically displayed in italics. `<p>This is <em>emphasized</em> text.</p>`

This is emphasized text.
`<mark>` Defines marked or highlighted text with a background color. `<p>Don't forget to buy <mark>milk</mark>.</p>`

Don't forget to buy milk.
`<small>` Defines smaller text. `<p>This is <small>smaller</small> text.</p>`

This is smaller text.
`<del>` Defines text that has been deleted from a document. Browsers usually strike a line through the text. `<p>My favorite color is <del>blue</del> red.</p>`

My favorite color is blue red.
`<ins>` Defines a text that has been inserted into a document. Browsers usually underline the text. `<p>My favorite color is blue <ins>red</ins>.</p>`

My favorite color is blue red.
`<sub>` Defines subscripted text (appears half a character below the normal line), commonly used for chemical formulas. `<p>The chemical formula for water is H<sub>2</sub>O.</p>`

The chemical formula for water is H2O.
`<sup>` Defines superscripted text (appears half a character above the normal line), commonly used for exponents or footnotes. `<p>E = mc<sup>2</sup>.</p>`

E = mc2.
`<u>` Defines text that is unarticulated and styled differently from normal text (e.g., misspelled words), typically underlined. `<p>This is <u>underlined</u> text.</p>`

This is underlined text.
`<s>` Defines text that is no longer correct or relevant (similar to `<del>`, but without the same semantic meaning as a content change). `<p>The price was <s>$100</s> $75.</p>`

The price was $100 $75.

Deprecated Tags (Not Supported in HTML5)

The following tags are no longer supported in HTML5. Use CSS for styling instead.

  • `<big>`: Used to make text one font size larger.
  • `<strike>`: Used to add a horizontal line through text (use `<del>` or `<s>` instead).
  • `<tt>`: Used for teletype or monospaced text (use CSS with `font-family: monospace;` or semantic tags like `<code>` or `<kbd>` instead).
  • `<font>`: Used to specify font size, color, and face.
